FuP schedule checker user guide:

  1. 1.Save standard schedule table (.xls file) in the local drive. The file should not have access restricted to a particular user. 

  2. 2.Export project schedule table from T3000 and save it the local drive. 

  3. 3.Use “main.exe” to open the application. Alternatively, “main.py” can be used if python is installed and all python dependencies (modules) are installed in the computer. 

  4. 4.Following window will open. Use “Select Program” menu to switch the program to “FuP schedule checker”. 

     
     

     

  5. 5.Following window will appear upon switching the program. The entry fields will be populated after browsing the required files. 

     
     

  6. 6.Browse and select the standard schedule file. (“.xls” file saved during S/N-1 of this document) 

  7. 7.Browse and select the project schedule file. (“.xls” file saved during S/N-2 of this document) 

  8. 8.Check that the entry fields are correctly filled and modify if needed. 

  9. 9.Press the button “Generate import file”. 

  10. 10.If the program runs successfully, the folder “result” will open automatically. Two files will be generated. Analyze the results before import in T3000. 

  • ➢“T3000_schedule_import.xls” – This file will be used to update T3000 FuP schedule. 

  • ➢“T3000_schedule_check.xls” – This file will have two additional columns “Remarks” and “Standard Seq” and can be used to check that the result is as desired. 

  1. Note: In both of the generated files, the FUPs which are not present in standard schedule will be highlighted in yellow.
